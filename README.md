# 연결 리스트 과제

## 소개

이 프로젝트는 C언어로 작성한 단순 연결 리스트 프로그램입니다.
사용자는 메뉴를 통해 정수를 추가, 삭제, 출력하고 리스트 크기를 확인할 수 있습니다.

## 파일 구성

- `main.c` : 메뉴와 사용자 입력 처리
- `Linkedlist.h` : 구조체 정의 및 함수 선언
- `Linkedlist.c` : 연결 리스트 기능 구현

## 기능

1. 맨 앞에 추가
2. 맨 뒤에 추가
3. 특정 위치에 추가
4. 특정 위치 삭제 및 삭제된 값 출력
5. 전체 목록 출력
6. 리스트 크기 확인
0. 메모리 해제 후 종료
